웹 개발 소개

웹 개발은 HTML, CSS, JavaScript를 사용하여 웹사이트를 구축하는 과정입니다.

웹 개발자는 사용자 경험(UX) 을 고려해 웹사이트를 만듭니다.

HTML 태그 학습

구조적 태그의 중요성

구조적 태그를 사용하면 웹 페이지의 의미와 구조가 명확해집니다.

이는 검색 엔진 최적화(SEO)와 접근성 향상에 도움이 됩니다.

작성일: 2025년 5월 1일

시맨틱 태그 예시

  • < header >- 페이지나 섹션의 헤더 부분
  • < nav > - 내비게이션 링크 모음
  • < main > - 문서의 주요 콘텐츠
  • < section > - 문서의 독립적인 섹션
  • < article > - 독립적인 콘텐츠 블록
  • < footer > - 페이지나 섹션의 푸터 부분
